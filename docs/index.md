# NEXUS Documentation

Free resources for learning SQL — whether you play the game or not.

---

## SQL Foundations Guide

**[Learn SQL Basics — A Beginner's Guide with Examples and Exercises](sql-foundations.md)**

Covers everything from SELECT to JOIN to GROUP BY, with plain-English
explanations, real-world analogies, syntax examples, and 16 practice
exercises. No jargon, no fluff.

**Topics covered:** SELECT, FROM, WHERE, COUNT, SUM, GROUP BY, ORDER BY,
JOIN, IN, HAVING, primary keys, foreign keys, aliases, subqueries,
NULL handling, SQL order of operations.

## The Game

**[NEXUS — Learn SQL by Solving a Mystery](../)**

A narrative-driven desktop game where you investigate corporate fraud
using real SQL queries. Season 1 teaches SQL fundamentals through 13
guided objectives across 6 investigation scenes.

Free. Open source. No prior SQL experience needed.

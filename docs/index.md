## SQL Foundations Guide

**[Learn SQL Basics — A Beginner's Guide with Examples and Exercises](sql-foundations.md)**

A 589-line standalone SQL reference — covers everything from SELECT to JOIN to GROUP BY,
with plain-English explanations, real-world analogies, syntax examples, and 16 practice
exercises across four difficulty tiers. You don't need to play NEXUS to use this guide.

**Topics covered:**
SELECT · FROM · WHERE · COUNT · SUM · GROUP BY · ORDER BY · JOIN · IN · HAVING ·
primary keys · foreign keys · aliases · subqueries · NULL handling · SQL order of operations

## The Game

**[NEXUS — Learn SQL by Solving a Mystery](https://michaelnocito.github.io/nexus-sql-mystery/)**

A narrative-driven desktop game where you investigate $1.87M in corporate fraud using
real SQL queries. Season 1 teaches SQL fundamentals through 16 guided objectives across
6 investigation scenes. Season 2 adds advanced SQL through a ghost-in-the-machine mystery.

Free. Open source. No prior SQL experience needed.
